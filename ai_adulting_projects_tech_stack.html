#1
Life Admin Assistant
Your AI personal organiser for adulting tasks

An AI agent that manages all day-to-day adult responsibilities — scheduling appointments, tracking deadlines, managing bills, handling assignments, sending reminders, and keeping life on track automatically.

Core Stack
Next.js 15 TypeScript FastAPI PostgreSQL Redis (caching) Supabase
AI / Agents
Claude claude-sonnet-4-20250514 LangGraph (agent flows) LlamaIndex (RAG) OpenAI Whisper (voice input)
Advanced / New Tech
Agentic AI (multi-step reasoning) Google Calendar MCP Gmail MCP integration Tool-use / Function Calling Notion API sync
#2
Wellness Manager
AI that tracks mood, health habits & mental wellbeing

Tracks mood patterns, sleep, exercise, water intake, and emotional state. Detects burnout signals, suggests routines, replaces fragmented health apps with one unified AI-powered dashboard.

Core Stack
React Native (mobile) Expo Node.js MongoDB InfluxDB (time-series data)
AI / Models
Sentiment analysis NLP Claude API (journal analysis) Llama 3.3 (on-device) TensorFlow Lite
Advanced / New Tech
Apple HealthKit / Google Fit API Wearable data (Fitbit/Whoop) Biometric pattern detection RAG over personal health history Voice journaling (Whisper)
#3
Finance Manager
AI personal finance advisor & budget tracker

Connects to bank accounts, categorises spending, tracks subscriptions, creates personalised budgets, flags unusual transactions, and gives proactive savings advice — like a financial advisor in your pocket.

Core Stack
Next.js 15 Prisma ORM PostgreSQL Python (analytics) Plaid API (banking)
AI / Models
Claude (spend insights) Pandas + scikit-learn Anomaly detection ML Prophet (expense forecasting)
Advanced / New Tech
Open Banking APIs Receipt OCR (Google Vision) Crypto wallet tracking AI-driven savings goals Natural language queries ("How much did I spend on food?")
